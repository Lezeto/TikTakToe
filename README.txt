This is a TikTakToe game. 
The whole page is a React component with TS, hosted locally with Vite, that simulates a webpage with multiple games.
Inside the App component is a TikTacToe component.
Both the TikTakToe and the Webpage are reactive.
The game allows you to play TikTakToe taking turns and tells you the winner. It also shows you the history of the current game and it can be restarted.

To run, go to the main directory and run
npm install
npm run dev
